# Truth-Lens: Verified Travel Memories

Truth-Lens is a minimalist travel journaling web application that helps users preserve authentic travel memories through GPS-verified check-ins. Unlike traditional travel platforms that rely on anonymous reviews and ratings, Truth-Lens focuses on creating a trustworthy digital travel passport where every memory is backed by proof of presence. Users can explore popular destinations, check in only when they are physically present at a location, upload a memorable photo, and write a short personal note about their experience. Each successful check-in is automatically added to their digital passport, creating a chronological record of the places they have visited.

The platform is designed with simplicity, authenticity, and privacy in mind. Location is accessed only when a user chooses to check in, ensuring that no continuous tracking occurs. Instead of lengthy reviews, users capture meaningful travel moments through concise memories and verified visits, making the platform more personal and reliable. Truth-Lens encourages exploration by transforming every verified visit into a lasting memory rather than another generic online review.

The website includes a clean and intuitive user interface consisting of a landing page, an explore section for discovering places, a GPS-based check-in page, a digital passport showcasing verified visits, and a user profile displaying travel history. The project is built using HTML, CSS, and JavaScript for the frontend, with future support planned for Supabase authentication, PostgreSQL database integration, image storage, and browser Geolocation APIs. As a first full-stack project, Truth-Lens demonstrates practical concepts such as responsive web design, geolocation, authentication, database connectivity, and user-centered product design while solving the problem of preserving genuine travel experiences in a simple and engaging way.
